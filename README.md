# Sortir.com

Premier projet s'inscrivant dans le cadre de notre formation Développeur Web & Web Mobile au sein de l'ENI Ecole Informatique, cette application a été développée par un groupe de 3 personnes en l'espace de deux semaines.

## 🚀 Objectif
Réaliser une application web Backend d'événementiel.

## 📖 Présentation du projet
En suivant un cahier des charges fourni par l'école, nous devions développer les fonctionnalités essentielles afin de répondre à la problèmatique demandée.

## 🛠 Technologies  et Logiciels utilisés

### Technologies :
- PHP
- Symfony
- Wampserver
- mySql
- HTML / CSS / Twig / JS

### Logiciels :
- IDE : PHPStorm
